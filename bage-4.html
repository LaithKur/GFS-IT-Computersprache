Welches sind die am häufigsten verwendeten Computersprache?

HTML

INFO

HTML bedeutet HyperText Markup Language.
HyperText: Verweist auf verlinkten Text, der es ermöglich, von einer Seite zur anderen zu navigieren.
Markup Language: Eine Sprache, die durch spezialle Tags(Auszeichnungen) den Aufbau und die Struktur von Documenten definiert(z.B. Überschriften, Absätze und Bilder).

CSS

INFO

CSS bedeutet Cascading Style Sheets:
Cascading = kaskadierend bedeutet, dass die stile in einer bestimmten Reihenfolge angewendet werden.
Style= Stil : bezieht sich aud das Design und die Formatierung von Elementen.
Sheets= Blätter : bezieht sich auf die CSS-Dateien, die die Stilregeln enthalten.
Also CSS ist eine Sprache zur Gestaltung und Formatierung von Webseiten.
